# Sénior+Tur — Barcelos com Alma

Website do projeto **Sénior+Tur**, desenvolvido no âmbito da Prova de Aptidão Profissional (PAP) do curso de Turismo.

## Sobre o projeto

A Sénior+Tur é uma proposta de agência de viagens focada no turismo sénior ativo, criando experiências culturais, turísticas e recreativas em Barcelos.
